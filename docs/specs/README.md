# SPEC 명세

각 SPEC은 착수 전 이 디렉토리에 `<SPEC-ID>.md`로 명세(목표·범위 In/Out·인수기준)를 먼저 적고 구현한다. 진행 상태와 시퀀싱은 [../../ROADMAP.md](../../ROADMAP.md)가 단일 진입점이다.

## 네이밍

- 파일명 = SPEC ID 그대로: `SPEC-AI-002.md`
- 본문 구성: 목표 / 범위(In·Out) / 인수기준 / 의존성 / 검증 방법

## 현황

`SPEC-AI-001` ~ `SPEC-AI-006`은 이 문서 구조가 도입되기 전 완료(DONE)됐다. 범위·인수 요약은 [../../ROADMAP.md](../../ROADMAP.md)의 Phase 1 표, as-built는 [../ARCHITECTURE.md](../ARCHITECTURE.md)·[../features/](../features/README.md) 참조. 이후 신규 SPEC부터 이 디렉토리에 명세를 둔다.
